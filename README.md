# spring-security-oauth2-social-login-2fa
Implementação completa de autenticação moderna com Java + Spring Security, incluindo login social com GitHub e Google, integração com OAuth 2.0 e OpenID Connect, além de autenticação de dois fatores (2FA) utilizando TOTP e QR Code. Fluxos OAuth, consumo de APIs externas, segurança avançada e integração com provedores de identidade.
